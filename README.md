## Preprint: *The MOND Acceleration Scale from Rindler–de Sitter Thermodynamics*  
*B. ten Broek* (March 2026)

Derives the MOND acceleration scale

$$
a_{0}= \frac{c\,H}{2\pi},

The manuscript introduces a unique ghost‑free covariant action and proves that the constant $a_{0}$ is a structural consequence of the vacuum.  

It is intended for an arXiv submission (gr‑qc) pending endorsement. Corrections are welcome.
